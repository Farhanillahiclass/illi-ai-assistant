# 🌌 ILLI OS | Neural Intelligence Interface

**ILLI OS** is a high-performance, local-first AI assistant for Windows. It features a cinematic **Ghost-Protocol HUD** designed with a "Natural Cyber" palette, providing real-time system automation and intelligent interaction without any paid API dependencies.

## 🚀 Features (No-API Required)

- **Neural Core**: Immersive Streamlit HUD with Glassmorphism and 3D animated orb.
- **Voice Command Center**: Offline Speech Recognition and high-quality TTS.
- **OS Command Center**: Deep hardware integration for process and file management.
- **Tactical Biometrics**: Real-time CPU, RAM, and Network energy bars.
- **Ghost Comms**: Native WhatsApp and Email automation.

## 🛠️ Tech Stack
- **Frontend**: Streamlit, Lottie, Custom CSS
- **Backend**: Python 3.13
- **System**: psutil, subprocess, os
- **Voice**: speech_recognition, pyttsx3

## 🔧 Installation

### Standard
```bash
pip install illi-ai-professional
```

### Developer Setup
```bash
git clone https://github.com/Farhanillahiclass/illi-ai-assistant.git
pip install -r requirements.txt
streamlit run ILLI_AI_PROFESSIONAL_FINAL.py
```
