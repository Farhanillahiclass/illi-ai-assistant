## 🛠️ Installation

### Quick Start (Recommended)

1. **Download the Repository**
   ```bash
   git clone https://github.com/Farhanillahiclass/illi-ai-assistant.git
   cd illi-ai-assistant
   ```

2. **Run the Professional Launcher**
   ```bash
   START_ILLI_PROFESSIONAL_FINAL.bat
   ```

### Manual Installation

1. **Install Python Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Install Additional Dependencies**
   ```bash
   pip install psutil speechrecognition pyttsx3 requests GitPython cryptography
   ```

3. **Run the Application**
   ```bash
   python ILLI_AI_PROFESSIONAL_FINAL.py
   ```
