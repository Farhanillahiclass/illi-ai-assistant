## 🎯 Step-by-Step Publishing Guide

### **Step 1: Create GitHub Repository**
1. Go to [GitHub](https://github.com)
2. Click **"New"** button (top right)
3. Repository name: `illi-ai-assistant`
4. Description: `Advanced AI Voice Assistant with Master Dashboard`
5. Choose **"Public"** (for anyone to use)
6. Click **"Create repository"**

### **Step 2: Prepare Local Repository**
```cmd
# Navigate to project directory
cd g:/Virtual Assistant

# Initialize Git repository
git init

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit - ILLI AI Assistant with Master Dashboard"
```

### **Step 3: Connect to GitHub**
```cmd
# Add remote repository (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/illi-ai-assistant.git

# Push to GitHub
git push -u origin main
```

## 🚀 Quick Start

### Installation
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/illi-ai-assistant.git

# Navigate to project
cd illi-ai-assistant

# Install dependencies
pip install -r requirements.txt
```

### Running ILLI

#### Voice Assistant
```bash
python illi_working_simple.py
```

#### Master Dashboard
```bash
python illi_dashboard_master.py
```

#### Easy Launchers (Windows)
```bash
# Voice Assistant
START_ILLI_WORKING_SIMPLE.bat

# Master Dashboard
START_ILLI_MASTER_DASHBOARD.bat
```
